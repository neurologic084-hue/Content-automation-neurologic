# Accounts the video app uses

All signed into with **Google** — use "Continue with Google" on each site.

## Paid

- Railway
- Submagic
- OpenRouter
- ElevenLabs
- Auphonic
- Blotato

## Free

- Cloudflare
- Supabase
- Pexels
- Tavily
- GitHub

---

If one runs out of credit, the app says which one and links to top it up. Most
keep working on a backup — only **OpenRouter** and **Submagic** stop things
entirely.
