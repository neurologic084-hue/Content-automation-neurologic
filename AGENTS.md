# Read TODO-JUNE.md first

`TODO-JUNE.md` in this directory is the current priority list, written by the
project owner. Read it at the start of any session here, before planning work.

If the task you have been given is on that list, follow its notes — several
entries record why a previous approach was removed, and re-adding it naively
reproduces bugs that took a day to find. If the task is NOT on that list, still
skim it: it explains what changed recently and what is deliberately absent
rather than broken.
